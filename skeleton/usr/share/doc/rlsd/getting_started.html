What is RLSD?

RLSD is a lightweight, single-user operating system.


What's special about RLSD?

RLSD ships with many classic, lightweight applications, adapted to run on modern hardware.


Where is the menu?

Right-click anywhere on the screen, press the RLSD button or ALT + F1.


How do I configure my wireless network card?

Go to the settings panel (through the Settings icon) and choose Wireless Wizard.


How do I enable remote access to RLSD?

Either enable Dropbear through the settings panel (for command-line access) or run x11vnc through the settings menu (for graphical access).


How do I install additional software?

Go to the settings panel and choose Package Manager.


What about security?

RLSD gives you the freedom to do whatever you want. However, this harms security, so critical system services or servers (e.g tftpd) are isolated from the rest of the system.

In addition, applications can be executed in a restricted environment (i.e so they cannot alter the system). If you mistrust an application or want to prevent it from reading your home directory, go to the menu and choose Run Restricted.


How do I access drives?

Go to the settings panel and choose Mounting Wizard.


Where can I find more documentation?

Fire up a file manager and go to /usr/share/doc. Each and every application included in RLSD is accompanied by its documentation, license, list of authors and version history.


Where can I connect with other users?

RLSD has an IRC channel: #rlsdlinux at irc.freenode.net. You can connect to it using X-Chat.
